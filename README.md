# Turbo Track Transportes

Site institucional da **Turbo Track Transportes Ltda - EPP**.

## Tecnologia
- Vue 3 + Vue Router + Vuex
- Vuetify 3
- Cores: #0C1A3E (azul) + #E8821A (laranja)

## Instalação
```bash
npm install
npm run serve
```

## Build
```bash
npm run build
```
